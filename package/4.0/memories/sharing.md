Sharing Memories in Reitti allows you to collaborate with others or let them view your travel stories without requiring them to have an account. You can share a Memory via a secure link, granting either view-only or edit access. This feature is perfect for sharing trip highlights with friends, family, or travel partners.

### How to Share a Memory
To share a Memory, follow these steps:

1. **Access the Memory**: Open the Reitti website and navigate to the Memories section. Select the Memory you want to share from your list.

2. **Open the Share Dialog**: In the Memory view, locate the header action bar at the top. Click the "Share" button to open the sharing dialog.

3. **Review What Will Be Shared**: The dialog will display a summary of what is included in the share:
   - The complete Memory with all its content blocks (e.g., text, maps, images).
   - Location data and maps for the Memory's time period.
   - Photos and text content within the Memory.
   - Trip and visit information during this time period.
   
   This ensures recipients see a full, self-contained version of your Memory.

4. **Select Access Level**: Choose between two options:
   - **View Only**: Recipients can read and view the Memory but cannot make changes.
   - **Edit Access**: Recipients can view and edit the Memory, allowing for collaborative storytelling.

5. **Set Expiration Duration**: After selecting the access level, choose how long the link will remain active. Options may include durations like 1 day, 1 week, 1 month, or never expire (depending on your settings).

6. **Generate the Link**: Click the confirmation button to create the share link. A success dialog will appear with the following details:
   - **Title**: "Share Link Created Successfully!"
   - **Memory Name**: The name of the shared Memory (e.g., "Visit to Spain 2025").
   - **Access Level**: Indicates whether it's "View Only" or "Edit Access".
   - **Share Link**: The generated link (e.g., a unique URL).
   - **How to Share**:
     - Copy the link above and send it to anyone you want to share with.
     - Recipients don't need an account to access the Memory.
     - The link will work according to the permissions you've set (view-only or edit).
     - Recipients can view and edit the Memory (if edit access is granted).

   Copy the link from this dialog and share it via email, messaging apps, or any other method.

### Managing Shared Links
To view, revoke, or manage the links you've created for a Memory:
- Navigate to the [Sharing configuration page](../configurations/share-access.md).
- Here, you can see a list of active shares and revoke access.

Sharing Memories is a great way to make your travels more interactive and memorable. If you have questions or encounter issues, feel free to reach out on [GitHub](https://github.com/dedicatedcode/reitti).
